This folder contains all the code which will help
put data inside the database. Make sure to review the
tables to check if it meets with the business logic.
To run the code, go on the Terminal (on top of vs code bar and
click new Terminal) write php follow by the file name
and the end of the file name put a dot and the php extension.
For Example: php setupOwner.php
Hit enter and table with data will be created inside the database.
